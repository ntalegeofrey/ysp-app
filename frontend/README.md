# Youth Supervision Platform - Next.js

Massachusetts Department of Youth Services (DYS) Youth Supervisory Platform

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Features

- Role-based access control (Admin, Manager, Technician, User)
- Resident and Staff Onboarding
- Staff Management
- AI Insights
- Analytics and Reporting
- Medication Count Management
- Incident Management
- Inventory Management
- Medical Run Coordination
- Fire Plan Management
- UCR (Unusual Client Report)
- Visitation and Phone Logs
- Sleep Log and Watch
- Log Book and Events Management
- Repair Management
- Contact Support

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Font Awesome Icons
